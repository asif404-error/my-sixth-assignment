# Project Name: DigiTools — Premium Digital Products Store

Welcome to my project. This is one of my assignments that I built as a part of my learning journey as a web developer. I'm still learning and growing, but I'm really glad and feeling proud of how this one turned out. DigiTools is a digital products marketplace where users can browse premium tools, add them to a cart, and manage their purchases — all in a clean and responsive UI.


## Technologies Used

-React.js: for building the UI components
-Tailwind CSS: for styling and responsive design
-DaisyUI: for UI component utilities
-JavaScript (ES6+): for all the logic and interactivity
-React-Toastify: for toast notifications (add to cart, remove, checkout)
-JSON: for managing the product data


### 3 Features

=> Product Browsing & Cart Management: Users can browse through 9 premium digital products displayed in a clean 3-column grid. Each product card shows the name, description, price, features, and a tag like Popular, New, or Best Seller. Clicking Buy Now adds the product to the cart instantly with a toast notification.

=> Products & Cart Toggle: There's a toggle button in the main section that lets users switch between the Products view and the Cart view without any page reload. The cart shows all selected items, their prices, a total, and a Proceed to Checkout button that clears everything at once.

=> Fully Responsive Design: The entire site is built mobile-first and looks great on all screen sizes — from phones to desktops. The navbar collapses into a hamburger menu on smaller screens, and the product grid stacks gracefully on mobile.


Final Statement: This project is part of my assignment and also a big step in my web development journey. I learned a lot building this — especially about component structure, state management, and making things look good with Tailwind.
